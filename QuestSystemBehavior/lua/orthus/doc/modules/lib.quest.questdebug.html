Module lib.quest.questdebug

Implements a debug mode for the quest system featuring visual quest trace on screen, advanced cheats and a simple shell for controlling quests while the map is in development.

Some cheats are disabled in Multiplayer!

Required modules:

  • qsb.oop
  • qsb.quest.questsystem
  • qsb.quest.questtools

ActivateDebugMode (_CheckQuests, _TraceQuests, _Cheats, _Console)
Activates the debug mode. Use the flags to decide which features you want to use.

Parameters:

  • _CheckQuests boolean Call debug check of behavior
  • _TraceQuests boolean Display quest status changes
  • _Cheats boolean Activate debug cheats
  • _Console boolean Activate debug shell

Usage:

    ActivateDebugMode(true, false, true, true);
